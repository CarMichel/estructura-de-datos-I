# Programa Heap Sort en Python

## 📖 Descripcion
Este programa implementa el algoritmo **Heap Sort** utilizando una interfaz grafica con **Tkinter**.  
Permite al usuario ingresar la cantidad de datos que desea ordenar y genera automaticamente los campos para introducir esos numeros.  
Luego, al presionar el boton correspondiente, aplica el algoritmo de **Heap Sort** y muestra la lista ordenada.

## 🛠️ Requisitos
- Python 3.x instalado en tu computadora.
- Libreria estandar `tkinter` (ya incluida en Python).

## 🚀 Ejecucion desde Git
1. Clona el repositorio:
    git clone https://github.com/CarMichel/estructura-de-datos-I.git
2. Entra a la carpeta donde se encuentra el archivo monticulo.py
3. Ejecuta el programa:
    python monticulo.py

## 🎯 Uso
Ingresa la cantidad de datos que quieres ordenar.
Se generaran los campos para introducir tus numeros.
Haz clic en el boton "Ordenar con Heap Sort".
El programa mostrara la lista ordenada en una ventana emergente.

## 📊 Aplicaciones
Aprender como funciona el algoritmo de Heap Sort.
Ordenar listas de numeros de manera eficiente.
Ejemplo practico de uso de Tkinter para interfaces graficas.
