# Programa Factorial en Python

## 📖 Descripcion
Este programa calcula el **factorial de un numero entero** ingresado por el usuario.  
El factorial se define como:
- `n! = n × (n-1) × (n-2) × ... × 1`
- Por ejemplo: `5! = 5 × 4 × 3 × 2 × 1 = 120`

## 🛠️ Requisitos
- Python 3.x instalado en tu computadora.

## 🚀 Ejecucion desde Git
1. Clona el repositorio:
   git clone https://github.com/tu_usuario/factorial_program.git
2. Entra a la carpeta donde se encuentra el archivo factorial.py
3. Ejecuta el programa:
   python factorial.py

## 🎯 Uso
El programa pedira que ingreses un numero entero.
Calcula el factorial y muestra el resultado en pantalla.
Si el numero es negativo, indica que el factorial no esta definido.
